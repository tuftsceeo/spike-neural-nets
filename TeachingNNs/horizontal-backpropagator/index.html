Activation functions

An activation function is applied to a neuron's weighted output before it becomes usable downstream. It introduces non-linearity so the network can learn complex patterns.

None No transformation. Output passes through unchanged (identity).
ReLU Sets negative values to zero. Fast and widely used in deep learning.
Leaky ReLU Like ReLU, but negative values get a small slope (×0.01) instead of a hard zero -- keeps a little gradient flowing even when the input is negative.
Sigmoid Squashes output to (0, 1). Good for probability outputs.
Tanh Squashes output to (−1, 1). Zero-centred, often better than Sigmoid.
Softplus Smooth approximation of ReLU. Always positive with no hard zero.
